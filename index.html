Setup

Rule implemented

During setup, the current player may remove only opponent-colour chains with zero liberties. If the current player has no legal capture, that pass is made automatically. Two consecutive setup passes start ordinary go. During scoring, players manually mark dead stone groups; those stones are removed only for scoring and added to the opponent's capture count.
